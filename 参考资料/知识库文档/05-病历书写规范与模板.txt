# 病历书写规范与模板（完整版）

## 1. 病历书写基本要求

### 1.1 书写原则
病历是医务人员在医疗活动过程中形成的文字、符号、图表、影像、切片等资料的总和。根据《病历书写基本规范》（卫生部2010年版），病历书写应遵循以下原则：
- 客观性：如实记录患者病情和诊疗过程
- 真实性：不得伪造、篡改病历内容
- 准确性：使用规范医学术语，数据准确
- 及时性：在规定时限内完成
- 完整性：包含所有必要项目

### 1.2 时限要求

| 病历类型 | 完成时限 | 责任人 |
|----------|---------|--------|
| 入院记录 | 24小时内 | 经治医师 |
| 首次病程记录 | 8小时内 | 经治医师 |
| 上级医师首次查房记录 | 48小时内 | 上级医师 |
| 日常病程记录（病危） | 每天 | 经治医师 |
| 日常病程记录（病重） | 至少3天1次 | 经治医师 |
| 日常病程记录（一般） | 至少5天1次 | 经治医师 |
| 阶段小结 | 超过30天时 | 经治医师 |
| 出院记录 | 24小时内 | 经治医师 |
| 死亡记录 | 24小时内 | 经治医师 |
| 死亡病例讨论 | 7天内 | 科主任主持 |

### 1.3 书写格式要求
- 使用蓝黑墨水或碳素笔书写（打印病历除外）
- 字迹工整，表述准确
- 使用中文和标准医学术语
- 应当使用规范的缩写（如：BP、HR、RR、T等）
- 修改时用双线划去，保留原记录可辨认，注明修改日期并签名

## 2. 入院记录

### 2.1 一般项目
```
姓名：XXX        性别：X        年龄：XX岁        民族：汉族
职业：XXXX      婚姻状况：XX    出生地：XX省XX市
入院日期：XXXX年XX月XX日XX时XX分
记录日期：XXXX年XX月XX日XX时XX分
病史陈述者：患者本人/家属（与患者关系：XX）
可靠程度：可靠/基本可靠/不可靠
```

### 2.2 主诉
主诉是促使患者就诊的主要症状（体征）及持续时间。

**书写要求：**
- 一般不超过20个字
- 包含症状和时间两个要素
- 不用诊断代替症状
- 时间应精确到天/小时

**示例：**
- "反复胸闷、气短2年，加重伴双下肢水肿1周"
- "上腹痛3天，黑便1天"
- "突发右侧肢体无力4小时"
- "咳嗽、咳痰伴发热5天"
- "发现血糖升高3年，视物模糊1个月"

### 2.3 现病史

现病史是入院记录的核心内容，按时间顺序描述疾病的发生、发展和诊疗经过。

#### 书写框架
1. **起病情况**：起病时间、缓急、诱因、初始症状
2. **主要症状特点**：部位、性质、程度、持续时间、加重/缓解因素
3. **伴随症状**：与主要症状相关或不相关的其他症状
4. **诊疗经过**：已做检查（结果）、已用药物（剂量、疗程、效果）
5. **一般情况变化**：精神、食欲、睡眠、大小便、体重变化

#### 示例（慢性心力衰竭急性加重）
```
患者2年前无明显诱因出现活动后胸闷、气短，登3层楼时明显，
休息约10分钟后可缓解，未予重视。此后症状逐渐加重，活动耐量
逐渐下降，登2层楼即出现症状。1周前受凉后症状明显加重，出现
夜间阵发性呼吸困难，需端坐位方可缓解，伴双下肢凹陷性水肿，
以踝部为著，晨轻暮重。尿量较前明显减少，约500-600ml/日。
院外口服"呋塞米20mg qd"，症状缓解不明显。
发病以来精神差，食欲减退，恶心，睡眠差（需高枕卧位），
大便正常，小便减少，体重近1周增加约3kg。
```

#### 示例（急性心肌梗死）
```
患者2小时前无明显诱因突发胸骨后压榨性疼痛，向左肩背部放射，
伴大汗、恶心、呕吐1次，为胃内容物，非喷射性。自含"硝酸甘油"
2片，症状无缓解。急呼120送入我院急诊。
发病以来精神差，未进食，大小便正常。
```

### 2.4 既往史

#### 书写内容
- **既往疾病史**：高血压（发现时间、最高血压、用药情况、控制情况）
- **糖尿病**（发现时间、用药情况、血糖控制情况）
- **冠心病**（发病时间、治疗经过）
- **传染病史**：肝炎（类型、治疗）、结核（治疗）、其他
- **手术外伤史**：手术名称、时间、术后恢复
- **输血史**：输血原因、时间、有无不良反应
- **预防接种史**：按计划完成情况
- **过敏史**：药物过敏（具体药物、过敏表现）、食物过敏、其他过敏

#### 示例
```
既往史：
高血压病史5年，最高血压180/110mmHg，长期口服"氨氯地平5mg qd"，
血压控制在140-150/85-90mmHg。
2型糖尿病史3年，口服"二甲双胍500mg bid"，空腹血糖7-8mmol/L。
否认肝炎、结核等传染病史。
否认手术外伤史。
否认输血史。
预防接种史随当地计划进行。
否认药物及食物过敏史。
```

### 2.5 个人史
- 出生地、长期居住地
- 职业及工作环境（有无毒物接触史）
- 吸烟史：XX年，XX支/日，已戒XX年（或未戒）
- 饮酒史：XX年，XXml/日（白酒/啤酒/红酒），已戒XX年
- 冶游史（必要时）
- 疫区居住/旅行史（必要时）

### 2.6 家族史
- 父母健康状况（是否健在，如已故，死亡原因）
- 兄弟姐妹健康状况
- 子女健康状况
- 家族遗传性疾病史（高血压、糖尿病、冠心病、肿瘤、遗传性疾病等）

## 3. 体格检查（详细版）

### 3.1 生命体征
```
T：XX°C    P：XX次/分    R：XX次/分    BP：XX/XXmmHg
身高：XXcm    体重：XXkg    BMI：XX kg/m²
```

### 3.2 一般检查

#### 发育与营养
- 发育：正常、异常（具体描述：侏儒症、巨人症等）
- 营养：良好（皮下脂肪充实）、中等（皮下脂肪适中）、不良（皮下脂肪菲薄）、消瘦（体重低于标准体重10%以上）、肥胖（BMI≥28kg/m²）

#### 神志与面容
- 神志：清楚（对答切题）、模糊（嗜睡，可唤醒）、谵妄（意识模糊伴躁动）、昏睡（强刺激可唤醒）、昏迷（浅昏迷GCS 9-12分/中昏迷GCS 5-8分/深昏迷GCS 3-4分）
- 面容：急性病容（痛苦表情）、慢性病容（面色晦暗）、贫血面容（面色苍白）、满月面容（面圆红润）、二尖瓣面容（双颊紫红）、粘液性水肿面容（面色苍黄、颜面浮肿）、苦笑面容（破伤风）

#### 体位
- 自动体位：患者可自如活动
- 被动体位：不能随意调整体位（昏迷、瘫痪）
- 强迫体位：
  - 强迫仰卧位：急性腹膜炎
  - 强迫俯卧位：脊柱疾病
  - 强迫侧卧位：胸膜疾病、大量胸腔积液
  - 强迫坐位（端坐呼吸）：心衰、呼吸困难
  - 强迫蹲位：先天性发绀型心脏病
  - 辗转体位：胆绞痛、肾绞痛
  - 角弓反张位：破伤风、脑膜炎

#### 皮肤黏膜
- 颜色：苍白（贫血）、发绀（缺氧）、黄染（黄疸，巩膜黄染TBIL>34.2μmol/L）、色素沉着（Addison病）、色素脱失（白癜风）
- 湿度：干燥（脱水）、潮湿（多汗）、盗汗（结核）
- 皮疹：斑疹（鲜红色不高出皮面）、丘疹（高出皮面）、斑丘疹（丘疹周围有红斑）、玫瑰疹（伤寒）、荨麻疹（过敏）、疱疹（水痘、带状疱疹）
- 出血点：瘀点（<2mm）、紫癜（3-5mm）、瘀斑（>5mm）
- 蜘蛛痣：肝脏疾病（雌激素灭活障碍），检查部位：上腔静脉分布区域
- 肝掌：大小鱼际发红，肝脏疾病
- 水肿：凹陷性（肾性、心源性、营养不良性）、非凹陷性（甲减、淋巴水肿）

#### 淋巴结检查
| 部位 | 检查方法 | 常见肿大原因 |
|------|---------|-------------|
| 颌下 | 头稍低，手指向内触摸 | 口腔/咽部感染 |
| 颈部 | 头偏向检查侧，放松胸锁乳突肌 | 淋巴瘤、转移癌、结核 |
| 锁骨上 | 坐位或站位 | 恶性肿瘤转移（Virchow淋巴结） |
| 腋窝 | 患者前臂放于检查者前臂 | 乳腺癌、上肢感染 |
| 滑车上 | 患者前臂稍屈 | 上肢感染、淋巴瘤 |
| 腹股沟 | 仰卧位 | 下肢感染、盆腔病变 |

淋巴结描述要素：部位、数目、大小（cm）、质地（软/韧/硬）、活动度（活动/固定）、压痛、局部皮肤

### 3.3 头颈部

#### 头颅
- 大小：正常、小头畸形、巨头
- 形态：方颅（佝偻病）、尖颅（Apert综合征）、巨颅（脑积水）
- 压痛：外伤、感染
- 包块：血肿、肿瘤
- 婴儿：前囟门（饱满/凹陷/隆起）、后囟门

#### 眼
- 眉毛：脱落（甲状腺功能减退、麻风）
- 睫毛：倒睫
- 眼睑：水肿（肾炎、心衰）、下垂（重症肌无力）、内眦赘皮
- 结膜：充血（结膜炎）、苍白（贫血）、出血点（感染性心内膜炎）
- 巩膜：黄染（黄疸）、蓝染（成骨不全）
- 瞳孔：大小（正常2-5mm）、形状（圆形/椭圆形/不规则）、对光反射（直接/间接）、调节反射
- 眼球：突出（甲亢、眶内肿瘤）、内陷（脱水）、运动（震颤-甲亢）

#### 耳
- 耳廓：畸形、结节（痛风石）
- 外耳道：分泌物（脓性-中耳炎、血性-外伤/肿瘤）
- 乳突：压痛（乳突炎）
- 听力：粗测（耳语试验）、音叉试验（Rinne试验、Weber试验）

#### 鼻
- 鼻外形：蛙鼻（鼻息肉）、鞍鼻（鼻骨破坏-梅毒/麻风）
- 鼻腔：通气（鼻塞）、分泌物（清涕-过敏、脓涕-感染）、出血
- 鼻窦：额窦、筛窦、上颌窦压痛（鼻窦炎）

#### 口咽
- 口唇：苍白（贫血）、发绀（缺氧）、皲裂（脱水）、疱疹（病毒感染）
- 口腔黏膜：充血（感染）、溃疡（口疮、白塞病）、色素沉着（Addison病）、Koplik斑（麻疹）
- 牙齿：龋齿、缺齿、义齿
- 牙龈：肿胀（增生-苯妥英/钙通道阻滞剂）、出血（维生素C缺乏、血小板减少）
- 舌：干燥（脱水）、舌苔（厚腻-感染/消化不良）、舌乳头萎缩（缺铁性贫血、维生素B缺乏）
- 咽部：充血（咽炎）、扁桃体肿大（I度不超过腭弓、II度超过腭弓、III度达到中线）、分泌物（脓性-化脓性扁桃体炎）
- 口腔气味：烂苹果味（DKA）、蒜臭味（有机磷中毒）、氨味（尿毒症）

#### 颈部
- 气管：居中/偏移（偏移提示气管移位）
- 甲状腺：
  - 视诊：肿大（I度看不见但可触及、II度既可触及又可见到、III度超过胸锁乳突肌外缘）
  - 触诊：大小、质地（软/韧/硬）、结节、压痛、震颤
  - 听诊：血管杂音（甲亢时可闻及）
- 颈静脉：怒张（心衰、心包填塞、上腔静脉综合征）——45°半卧位观察
- 颈动脉：搏动（增强-主动脉瓣关闭不全、减弱-主动脉缩窄）

### 3.4 胸部

#### 肺部检查
- **视诊**：呼吸运动（频率、节律、深度）、肋间隙（增宽-肺气肿/气胸、变窄-胸膜粘连）
- **触诊**：语颤（增强-肺实变/肺不张、减弱-气胸/胸腔积液/肺气肿）、胸廓扩张度（不对称-单侧病变）、胸膜摩擦感
- **叩诊**：清音（正常）、浊音（实变/积液）、实音（大量积液）、过清音（肺气肿）、鼓音（气胸）
- **听诊**：呼吸音（正常/减弱/增强）、啰音（干性-支气管炎/哮喘、湿性-肺炎/心衰/支扩）、胸膜摩擦音、语音共振

#### 心脏检查
- **视诊**：心尖搏动位置（正常：第5肋间左锁骨中线内0.5-1cm）、心前区隆起（儿童期心脏病）
- **触诊**：心尖搏动（抬举性搏动-左室肥大）、震颤（器质性心脏病）、心包摩擦感
- **叩诊**：心界大小（左界：第2肋间2-3cm、第3肋间2.5-3.5cm、第4肋间3.5-4.5cm、第5肋间7-9cm；右界：第2肋间2-3cm、第3肋间3-4cm、第4肋间5-6cm）
- **听诊**：
  - 心率：正常60-100次/分
  - 心律：齐/不齐
  - 心音：S1（二尖瓣/三尖瓣关闭）、S2（主动脉瓣/肺动脉瓣关闭）
  - 额外心音：S3奔马律（心衰）、S4奔马律（左室顺应性下降）、开瓣音（二尖瓣狭窄）、心包叩击音（缩窄性心包炎）
  - 杂音：时期（收缩期/舒张期/连续性）、性质（吹风样/隆隆样/机器样/乐音样）、强度（1-6级）、传导方向

#### 心脏杂音鉴别表

| 疾病 | 杂音时期 | 杂音部位 | 性质 | 传导 | 特征 |
|------|---------|---------|------|------|------|
| 二尖瓣狭窄 | 舒张中晚期 | 心尖部 | 隆隆样 | 局限 | 局部、左侧卧位更清楚 |
| 二尖瓣关闭不全 | 全收缩期 | 心尖部 | 吹风样 | 左腋下 | 响亮、粗糙 |
| 主动脉瓣狭窄 | 收缩期 | 胸骨右缘2肋间/左缘3肋间 | 喷射性 | 颈部 | 递增递减型 |
| 主动脉瓣关闭不全 | 舒张早期 | 胸骨左缘3-4肋间 | 叹气样 | 心尖部 | 递减型 |
| 室间隔缺损 | 全收缩期 | 胸骨左缘3-4肋间 | 粗糙 | 广泛 | 常伴震颤 |
| 动脉导管未闭 | 连续性 | 胸骨左缘2肋间 | 机器样 | 局限 | 收缩期增强 |

### 3.5 腹部

#### 视诊
- 外形：平坦（正常）、膨隆（腹水、肠胀气、腹腔肿瘤）、凹陷（脱水、消瘦）
- 腹壁静脉曲张：门脉高压（以脐为中心放射状）、下腔静脉阻塞（两侧腹壁）、上腔静脉阻塞（上腹部）
- 胃肠型及蠕动波：幽门梗阻（胃型+蠕动波）、肠梗阻（肠型+蠕动波）
- 腹纹：妊娠纹、紫纹（Cushing综合征）
- 疝：腹股沟疝、脐疝、切口疝

#### 触诊
- 压痛：部位（定位诊断价值大）
  - 右上腹：胆囊炎、肝炎、胆石症
  - 右下腹：阑尾炎（McBurney点）
  - 上腹部：胃炎、消化性溃疡、胰腺炎
  - 脐周：小肠病变
  - 下腹部：膀胱炎、盆腔炎
- 反跳痛：腹膜刺激征（提示腹膜炎）
- 肌紧张：腹膜刺激征
- 肝触诊：大小（正常肋下触及不到/触及<1cm）、质地（软如唇/韧如鼻/硬如额）、边缘（锐/钝）、表面（光滑/结节状）、压痛、搏动
- 脾触诊：大小（正常不可触及）、测量（I线：左锁骨中线肋缘下、II线：左锁骨中线与脐之间、III线：脐与髂前上棘之间）
- 肾触诊：双手触诊法
- 液波震颤：腹水量>3000-4000ml时阳性

#### 叩诊
- 肝浊音界：正常位于右锁骨中线第5肋间至肋缘
- 移动性浊音：腹水量>1000ml阳性
- 膀胱叩诊：耻骨上方浊音区
- 脾浊音区：Traube区（左腋中线第9-11肋间）

#### 听诊
- 肠鸣音：正常4-5次/分
  - 亢进：>10次/分（机械性肠梗阻、急性胃肠炎）
  - 减弱：明显减少（低钾血症、腹膜炎）
  - 消失：持续3-5分钟无肠鸣音（麻痹性肠梗阻、弥漫性腹膜炎）
- 血管杂音：腹主动脉瘤（收缩期杂音）、肾动脉狭窄（上腹部两侧）

### 3.6 脊柱四肢

#### 脊柱
- 弯曲度：正常颈曲前凸、胸曲后凸、腰曲前凸
- 侧弯：脊柱侧凸症
- 压痛：棘突压痛（椎体病变）、棘突旁压痛（椎间盘突出）
- 叩击痛：肾区叩击痛（肾盂肾炎、肾结石）
- 活动度：前屈（正常90°）、后伸（30°）、侧屈（20-30°）、旋转（30-45°）

#### 四肢
- 关节：肿胀、压痛、活动受限、畸形
- 杵状指（趾）：甲床角度消失（>180°），见于肺癌、支气管扩张、感染性心内膜炎、发绀型先心病
- 匙状甲（反甲）：缺铁性贫血
- 水肿：凹陷性（心源性、肾源性、肝源性、营养不良性）、非凹陷性（淋巴水肿、粘液性水肿）
- 畴形：膝内翻（O型腿）、膝外翻（X型腿）、足内翻、足外翻
- 静脉曲张：大隐静脉曲张、小隐静脉曲张

### 3.7 神经系统

#### 颅神经（II-XII）
| 编号 | 名称 | 检查方法 | 异常表现 |
|------|------|---------|---------|
| I | 嗅神经 | 分别测试两侧嗅觉 | 嗅觉减退/消失 |
| II | 视神经 | 视力、视野、眼底 | 视力下降、视野缺损 |
| III | 动眼神经 | 眼球运动、瞳孔 | 上睑下垂、眼球运动障碍 |
| IV | 滑车神经 | 眼球向下内运动 | 下视困难 |
| V | 三叉神经 | 面部感觉、咀嚼肌力 | 面部感觉障碍 |
| VI | 外展神经 | 眼球外展 | 内斜视 |
| VII | 面神经 | 额纹、闭眼、示齿 | 周围性/中枢性面瘫 |
| VIII | 听神经 | 听力（Rinne/Weber） | 耳聋、耳鸣 |
| IX/X | 舌咽/迷走 | 悬雍垂、咽反射、吞咽 | 构音障碍、吞咽困难 |
| XI | 副神经 | 转头、耸肩 | 转头耸肩无力 |
| XII | 舌下神经 | 伸舌 | 舌偏斜、萎缩 |

#### 运动功能
- 肌力分级（0-5级）：
  - 0级：完全瘫痪，无肌肉收缩
  - 1级：可见肌肉收缩，无关节运动
  - 2级：肢体可在床面移动，不能抬起
  - 3级：肢体能抬离床面
  - 4级：能做抗阻力运动，但力量差
  - 5级：正常肌力
- 肌张力：正常、增高（痉挛性-锥体束、强直性-锥体外系）、降低
- 不随意运动：震颤（静止性-帕金森、意向性-小脑）、手足徐动、舞蹈样运动

#### 感觉功能
- 浅感觉：痛觉（针刺）、触觉（棉签）、温度觉（冷热试管）
- 深感觉：位置觉（关节位置）、振动觉（音叉）、运动觉
- 复合感觉：皮肤定位觉、两点辨别觉、实体辨别觉

## 7. 鉴别诊断书写范例

### 7.1 以"胸痛"为主题的鉴别诊断

```
鉴别诊断：
1. 急性冠脉综合征（ACS）：
   - 支持点：胸骨后压榨性疼痛，向左肩放射，伴大汗
   - 不支持点：疼痛非持续性，硝酸甘油可部分缓解，cTnI正常
   - 进一步检查：动态复查心电图、心肌酶谱

2. 主动脉夹层：
   - 支持点：突发剧烈胸痛，向背部放射
   - 不支持点：无撕裂样疼痛性质，双上肢血压对称，D-dimer正常
   - 进一步检查：主动脉CTA

3. 急性肺栓塞：
   - 支持点：突发胸痛伴呼吸困难，有下肢DVT病史
   - 不支持点：无咯血，D-dimer正常，血气分析正常
   - 进一步检查：CTPA（必要时）

4. 急性心包炎：
   - 支持点：胸痛与呼吸相关，前倾位可缓解
   - 不支持点：无胸膜摩擦音，ECG无广泛ST段抬高
   - 进一步检查：心脏超声

5. 气胸：
   - 支持点：突发胸痛伴呼吸困难
   - 不支持点：双肺呼吸音对称，无叩诊鼓音
   - 进一步检查：胸部X线

6. 肋间神经痛：
   - 支持点：沿肋间分布的刺痛
   - 不支持点：疼痛非刺痛性质，无触痛
   - 进一步检查：排除器质性疾病
```

### 7.2 以"头痛"为主题的鉴别诊断

```
鉴别诊断：
1. 偏头痛：
   - 支持点：反复发作性搏动性头痛，伴恶心
   - 不支持点：无先兆症状，疼痛部位不典型
   - 进一步检查：头颅MRI排除器质性病变

2. 紧张型头痛：
   - 支持点：双侧压迫性头痛，持续性
   - 不支持点：疼痛程度较重，伴视觉症状
   - 进一步检查：评估触发因素

3. 蛛网膜下腔出血：
   - 支持点：突发剧烈头痛（"有生以来最严重的头痛"）
   - 不支持点：无意识障碍，无脑膜刺激征，CT未见出血
   - 进一步检查：腰穿查脑脊液

4. 颅内占位性病变：
   - 支持点：进行性加重头痛，晨起为著
   - 不支持点：无呕吐、视乳头水肿
   - 进一步检查：头颅MRI增强

5. 高血压脑病：
   - 支持点：血压显著升高伴头痛
   - 不支持点：血压控制尚可，无意识障碍
   - 进一步检查：监测血压、头颅MRI
```

```

2. 急性前壁ST段抬高型心肌梗死

手术经过：
1. 患者平卧位，消毒铺巾，局部麻醉。
2. 穿刺右侧桡动脉，置入6F动脉鞘管。
3. 送入J形导丝至升主动脉，沿导丝送入6F JL4指引导管至左冠状动脉开口。
4. 造影示：前降支近段狭窄95%，TIMI血流2级；回旋支、右冠状动脉未见明显狭窄。
5. 经指引导管送入Runthrough导丝通过前降支病变至远端。
6. 沿导丝送入球囊（2.5×20mm）于病变处预扩张（12atm×15s×2次）。
7. 沿导丝送入药物洗脱支架（3.0×28mm）至病变处，定位准确后释放（14atm×20s）。
8. 支架释放后造影复查：前降支近段残余狭窄<10%，TIMI血流3级。
9. 撤出导管及导丝，压迫器压迫穿刺点。
10. 手术顺利，术中出血少，约5ml。
11. 术中患者生命体征平稳，无特殊不适。

术中用药：
肝素5000U，硝酸甘油200μg冠脉内注射

术中出血量：约5ml
术中输血：无
术中并发症：无

术者签名：XXX
日期：XXXX年XX月XX日
```
