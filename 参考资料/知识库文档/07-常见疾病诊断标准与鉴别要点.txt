# 常见疾病诊断标准与鉴别要点（完整版）

## 1. 高血压

### 1.1 诊断标准（2018年中国高血压防治指南）

| 分级 | 收缩压(mmHg) | 舒张压(mmHg) |
|------|--------------|--------------|
| 正常血压 | <120 | <80 |
| 正常高值 | 120-139 | 80-89 |
| 1级高血压（轻度） | 140-159 | 90-99 |
| 2级高血压（中度） | 160-179 | 100-109 |
| 3级高血压（重度） | ≥180 | ≥110 |

**诊断要求：** 非同日3次测量血压均升高（收缩压≥140mmHg和/或舒张压≥90mmHg）

### 1.2 高血压危险分层

#### 危险因素
- 年龄：男性>55岁，女性>65岁
- 吸烟
- 血脂异常：TC≥5.7mmol/L或LDL-C≥3.4mmol/L或HDL-C<1.0mmol/L
- 空腹血糖：6.1-6.9mmol/L和/或糖负荷后2h血糖：7.8-11.0mmol/L
- 腹型肥胖：腰围男性≥90cm，女性≥85cm
- 早发心血管病家族史（一级亲属发病年龄<50岁）
- 缺乏体力活动
- 高敏C反应蛋白≥3mg/L

#### 靶器官损害
- 左心室肥厚（心电图或超声心动图）
- 颈动脉内中膜增厚（IMT≥0.9mm）或斑块
- 踝臂指数（ABI）<0.9
- 估算肾小球滤过率（eGFR）降低：30-59ml/(min·1.73m²)
- 微量白蛋白尿：30-300mg/24h

#### 并发症
- 脑血管病：脑出血、缺血性脑卒中、短暂性脑缺血发作
- 心脏疾病：心肌梗死、心绞痛、冠脉血运重建、慢性心力衰竭
- 肾脏疾病：糖尿病肾病、肾功能不全（eGFR<30）
- 血管疾病：主动脉夹层、外周动脉疾病
- 高血压性视网膜病变：出血、渗出、视乳头水肿

### 1.3 治疗原则

#### 生活方式干预
- 限盐：每日食盐<6g
- 控制体重：BMI<24kg/m²，腰围男性<90cm，女性<85cm
- 规律运动：每周≥150分钟中等强度有氧运动
- 戒烟限酒
- 心理平衡

#### 药物治疗
**常用降压药物：**

| 类别 | 代表药物 | 适应证 | 禁忌证 |
|------|---------|--------|--------|
| ACEI | 依那普利、贝那普利 | 心衰、糖尿病肾病 | 妊娠、高钾血症 |
| ARB | 缬沙坦、厄贝沙坦 | 心衰、糖尿病肾病 | 妊娠、高钾血症 |
| CCB | 氨氯地平、硝苯地平 | 老年、单纯收缩期高血压 | 心衰 |
| 利尿剂 | 氢氯噻嗪、吲达帕胺 | 老年、盐敏感性高血压 | 痛风 |
| β受体阻滞剂 | 美托洛尔、比索洛尔 | 心衰、心梗后 | 哮喘、心动过缓 |

**降压目标：**
- 一般患者：<140/90mmHg
- 糖尿病/肾病：<130/80mmHg
- 老年患者（≥65岁）：<150/90mmHg（可耐受则<140/90mmHg）

### 1.4 鉴别诊断

| 需鉴别疾病 | 鉴别要点 | 关键检查 |
|-----------|---------|---------|
| 继发性高血压（肾性） | 年轻发病、低钾、腹部血管杂音 | 肾动脉超声、肾功能 |
| 原发性醛固酮增多症 | 高血压+低钾+碱中毒 | 血浆醛固酮/肾素比值 |
| 嗜铬细胞瘤 | 阵发性高血压+心悸+多汗 | 血尿儿茶酚胺、CT |
| 库欣综合征 | 满月脸+向心性肥胖+高血压 | 皮质醇节律、地塞米松抑制试验 |
| 主动脉缩窄 | 上肢血压显著高于下肢 | 主动脉CTA |

## 2. 糖尿病

### 2.1 诊断标准（2020年中国2型糖尿病防治指南）

| 指标 | 正常 | 糖尿病 | 糖尿病前期 |
|------|------|--------|-----------|
| 空腹血糖 | <6.1mmol/L | ≥7.0mmol/L | 6.1-6.9mmol/L |
| OGTT 2h | <7.8mmol/L | ≥11.1mmol/L | 7.8-11.0mmol/L |
| 随机血糖 | <11.1mmol/L | ≥11.1mmol/L | - |
| HbA1c | <6.0% | ≥6.5% | 6.0-6.4% |

**诊断要求：** 无症状者需两次检测异常（可相同或不同指标）

### 2.2 糖尿病分型

#### 1型糖尿病
- 发病机制：自身免疫性β细胞破坏
- 特点：多见于青少年，起病急，症状明显，胰岛素绝对缺乏
- 自身抗体：GAD抗体、IA-2抗体、ZnT8抗体阳性
- 治疗：终身胰岛素治疗
- 急性并发症：糖尿病酮症酸中毒（DKA）

#### 2型糖尿病
- 发病机制：胰岛素抵抗为主伴胰岛素分泌不足
- 特点：多见于中老年人，起病隐匿，症状不明显
- 治疗：生活方式干预+口服降糖药±胰岛素
- 急性并发症：高渗高血糖综合征

#### 其他特殊类型
- 妊娠期糖尿病（GDM）
- 胰腺疾病所致糖尿病
- 内分泌疾病所致糖尿病
- 药物所致糖尿病

### 2.3 治疗目标

| 指标 | 目标值 |
|------|--------|
| 空腹血糖 | 4.4-7.0mmol/L |
| 餐后2h血糖 | <10.0mmol/L |
| HbA1c | <7.0% |
| 血压 | <130/80mmHg |
| LDL-C | <2.6mmol/L（无ASCVD）<br><1.8mmol/L（有ASCVD） |

### 2.4 急性并发症

#### 糖尿病酮症酸中毒（DKA）
- 诊断：血糖>13.9mmol/L，pH<7.3，血酮>5mmol/L
- 临床表现：多饮多尿加重、恶心呕吐、腹痛、Kussmaul呼吸、烂苹果味
- 治疗：补液、小剂量胰岛素、纠正电解质

#### 高渗高血糖综合征（HHS）
- 诊断：血糖>33.3mmol/L，血浆渗透压>320mOsm/L，无明显酮症
- 特点：多见于老年2型糖尿病，脱水严重，意识障碍明显
- 治疗：大量补液、胰岛素、纠正电解质

### 2.5 实验室检查指标详解

| 检查项目 | 正常参考值 | 临床意义 |
|---------|-----------|---------|
| 空腹血糖 | 3.9-6.1mmol/L | 糖代谢核心指标 |
| OGTT 2h | <7.8mmol/L | 诊断糖尿病前期/糖尿病 |
| HbA1c | <6.0% | 反映近2-3月平均血糖 |
| 空腹胰岛素 | 2.6-24.9μU/mL | 评估胰岛功能 |
| C肽 | 0.3-1.3ng/mL | 反映内源性胰岛素分泌 |
| 血脂四项 | TC<5.2, TG<1.7, LDL<3.4, HDL>1.0 | 心血管风险评估 |
| 尿微量白蛋白 | <30mg/24h | 早期肾损害 |

### 2.6 鉴别诊断

| 需鉴别疾病 | 鉴别要点 | 关键检查 |
|-----------|---------|---------|
| 1型vs2型糖尿病 | 年龄、BMI、C肽、抗体 | GAD抗体、C肽释放试验 |
| 妊娠期糖尿病 | 妊娠24-28周筛查 | OGTT |
| 甲亢性糖尿病 | 甲亢症状+血糖升高 | 甲功 |
| 肾性糖尿 | 血糖正常、尿糖阳性 | 肾功能、血糖 |
| 库欣综合征 | 向心性肥胖+高血糖 | 皮质醇 |

## 3. 冠心病

### 3.1 诊断要点

#### 稳定型心绞痛
- 典型胸骨后压榨性疼痛
- 劳力或情绪激动诱发，休息或含服硝酸甘油缓解
- 持续3-5分钟
- 发作时心电图ST段压低≥0.1mV

#### 不稳定型心绞痛/非ST段抬高型心肌梗死（ACS）
- 静息性心绞痛（>20分钟）
- 初发心绞痛（1个月内）
- 恶化性心绞痛（加重、持续时间延长）
- 心肌酶谱/肌钙蛋白升高（NSTEMI）

#### ST段抬高型心肌梗死（STEMI）
- 持续胸痛>30分钟
- 心电图ST段弓背向上抬高
- 心肌酶谱/肌钙蛋白升高
- 病理性Q波

### 3.2 危险因素
- 不可改变：年龄、性别、家族史
- 可改变：高血压、高血脂、糖尿病、吸烟、肥胖、缺乏运动
- 新兴危险因素：hsCRP、同型半胱氨酸、Lp(a)

### 3.3 治疗
- 抗血小板：阿司匹林+氯吡格雷/替格瑞洛
- 抗凝：肝素/低分子肝素
- 他汀类：强化降脂
- β受体阻滞剂：降低心肌耗氧
- ACEI/ARB：心室重构
- STEMI：急诊PCI（D2B<90分钟）或溶栓（发病<12小时）

### 3.4 影像学表现

| 检查方法 | 典型表现 |
|---------|---------|
| 心电图 | ST段压低/抬高、T波倒置、病理性Q波 |
| 心脏超声 | 室壁运动异常、EF降低、室壁瘤 |
| 冠脉CTA | 冠脉狭窄、钙化积分 |
| 冠脉造影 | 金标准，明确狭窄部位和程度 |
| 心肌灌注显像 | 缺血/梗死区域 |

### 3.5 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 急性心包炎 | 胸痛与呼吸相关、前倾位缓解、ECG广泛ST抬高 |
| 主动脉夹层 | 撕裂样疼痛、双上肢血压差、CTA确诊 |
| 肺栓塞 | 呼吸困难、D-dimer升高、CTPA |
| 气胸 | 突发呼吸困难、患侧呼吸音减弱 |
| 肋间神经痛 | 沿肋间走行的刺痛、压痛 |
| 胃食管反流 | 烧灼感、与进食相关、抗酸药缓解 |

## 4. 肺炎

### 4.1 社区获得性肺炎（CAP）诊断标准
1. 新出现的咳嗽、咳痰或原有呼吸道症状加重，伴或不伴脓痰/胸痛/呼吸困难/咯血
2. 发热
3. 肺实变体征和/或湿啰音
4. WBC>10×10⁹/L或<4×10⁹/L，伴或不伴核左移
5. 胸部影像学：新出现的浸润性病变

### 4.2 CURB-65评分

| 指标 | 标准 | 分值 |
|------|------|------|
| C（意识障碍） | 新发意识模糊（对人物、地点、时间的定向障碍） | 1 |
| U（尿素氮） | >7mmol/L（>19.6mg/dl） | 1 |
| R（呼吸频率） | ≥30次/分 | 1 |
| B（血压） | 收缩压<90mmHg或舒张压≤60mmHg | 1 |
| 65（年龄） | ≥65岁 | 1 |

**评分意义：**
- 0-1分：低危，门诊治疗
- 2分：中危，考虑住院
- 3-5分：高危，住院治疗，3-4分考虑ICU

### 4.3 病原学诊断
- 痰培养+药敏
- 血培养（重症）
- 肺炎链球菌尿抗原
- 军团菌尿抗原
- 呼吸道病毒检测

### 4.4 影像学表现

| 肺炎类型 | 影像学特征 |
|---------|-----------|
| 大叶性肺炎 | 大片实变影、支气管充气征 |
| 支气管肺炎 | 沿支气管分布的斑片影 |
| 间质性肺炎 | 网状或磨玻璃影 |
| 空洞性肺炎 | 空洞形成、液平 |
| 胸腔积液 | 肋膈角变钝或大量积液 |

### 4.5 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 肺结核 | 低热、盗汗、消瘦、痰涂片抗酸杆菌阳性 |
| 肺癌 | 体重下降、痰中带血、CT占位 |
| 肺栓塞 | 突发呼吸困难、D-dimer升高 |
| 心衰肺水肿 | 端坐呼吸、BNP升高、心影增大 |
| 肺间质纤维化 | 慢性进行性呼吸困难、HRCT蜂窝样改变 |

## 5. 急性阑尾炎

### 5.1 诊断要点
- **转移性右下腹痛**（最典型表现，70-80%）：先脐周或上腹痛，数小时后转移至右下腹
- 恶心、呕吐（60-80%）
- 低热（37.5-38.5°C）
- 右下腹压痛（McBurney点：脐与右髂前上棘连线中外1/3交界处）
- 反跳痛、肌紧张（腹膜刺激征）
- 结肠充气试验、腰大肌试验、闭孔内肌试验可阳性

### 5.2 分型

| 类型 | 病理特点 | 临床表现 |
|------|---------|---------|
| 急性单纯性 | 黏膜层炎症 | 症状较轻 |
| 急性化脓性 | 全层炎症 | 症状明显 |
| 急性坏疽性 | 坏死穿孔 | 重症表现 |
| 阑尾周围脓肿 | 大网膜包裹 | 右下腹包块 |

### 5.3 Alvarado评分

| 指标 | 分值 |
|------|------|
| 转移性右下腹痛 | 1分 |
| 厌食 | 1分 |
| 恶心/呕吐 | 1分 |
| 右下腹压痛 | 2分 |
| 反跳痛 | 1分 |
| 发热（>37.3°C） | 1分 |
| WBC升高 | 2分 |
| 中性粒细胞比例升高 | 1分 |
| **总分≥7分：高度可能阑尾炎** | |

### 5.4 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 卵巢囊肿蒂扭转 | 突发右下腹痛，B超可发现附件包块 |
| 宫外孕破裂 | 停经史，腹痛+阴道出血，β-hCG阳性 |
| 右输尿管结石 | 阵发性绞痛，放射至会阴，尿常规见红细胞 |
| 肠系膜淋巴结炎 | 多见于儿童，常有上呼吸道感染史 |
| 克罗恩病 | 慢性病程，腹泻，肠镜可确诊 |
| 急性肠系膜淋巴结炎 | 儿童多见，上感后右下腹痛 |
| 盲肠憩室炎 | 类似阑尾炎，CT可鉴别 |

## 6. 脑卒中

### 6.1 缺血性脑卒中（脑梗死）

#### 诊断要点
- 突发局灶性神经功能缺损（偏瘫、失语、感觉障碍等）
- 症状持续>24小时
- 头颅CT排除脑出血
- 头颅MRI（DWI）显示梗死灶

#### 治疗时间窗

| 治疗方法 | 时间窗 | 适应证 |
|----------|--------|--------|
| 静脉溶栓（rt-PA） | 发病4.5小时内 | 无禁忌证 |
| 静脉溶栓（尿激酶） | 发病6小时内 | 无禁忌证 |
| 动脉取栓 | 发病6-24小时内 | 大血管闭塞 |

#### 溶栓禁忌证
- 近3个月内有重大外伤或手术
- 近3周内有消化道出血
- 近2周内有大型手术
- 血小板<100×10⁹/L
- 收缩压>185mmHg或舒张压>110mmHg
- 血糖<2.7mmol/L
- CT显示出血或大面积低密度影

### 6.2 出血性脑卒中（脑出血）

#### 诊断要点
- 突发头痛、呕吐、意识障碍
- 头颅CT显示高密度影
- 脑膜刺激征阳性（蛛网膜下腔出血）
- 血压明显升高

#### 治疗原则
- 控制血压（收缩压目标<140mmHg）
- 降低颅内压（甘露醇、高渗盐水）
- 止血（氨甲环酸，争议）
- 必要时手术（小脑出血>3cm、脑疝先兆等）

### 6.3 FAST识别法
- **F**ace（面部）：微笑时面部不对称
- **A**rm（手臂）：双臂平举时一侧下垂
- **S**peech（言语）：说话含糊不清
- **T**ime（时间）：立即拨打急救电话

### 6.4 NIHSS评分要点

| 评估项目 | 评分范围 | 说明 |
|---------|---------|------|
| 意识水平 | 0-3 | 从清醒到昏迷 |
| 凝视 | 0-2 | 正常到凝视麻痹 |
| 视野 | 0-3 | 从无缺损到全盲 |
| 面瘫 | 0-3 | 正常到完全瘫痪 |
| 上肢运动 | 0-4 | 从无下落到无运动 |
| 下肢运动 | 0-4 | 从无下落到无运动 |
| 肢体共济失调 | 0-2 | 从无到双侧 |
| 感觉 | 0-2 | 从无到严重 |
| 语言 | 0-3 | 从无失语到失语 |
| 构音障碍 | 0-2 | 从无到严重 |
| 忽视 | 0-2 | 从无到严重 |
| **总分42分，≥15分为重度卒中** | | |

### 6.5 影像学表现

| 检查 | 脑梗死表现 | 脑出血表现 |
|------|-----------|-----------|
| CT（早期） | 可能阴性或灰白质界限模糊 | 高密度影（白色） |
| CT（24h后） | 低密度影 | 周围水肿、占位效应 |
| MRI-DWI | 早期即显示高信号（金标准） | T1高信号、T2低信号 |
| MRA | 血管闭塞/狭窄 | 无直接血管征象 |
| CTA | 血管闭塞部位 | 无直接血管征象 |

### 6.6 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 低血糖 | 血糖<2.8mmol/L、补糖后恢复 |
| 颅内肿瘤 | 慢性进行性加重、CT/MRI占位 |
| 硬膜下血肿 | 外伤史、CT硬膜下高密度 |
| 癫痫发作后 | Todd麻痹、既往癫痫史 |
| 偏头痛先兆 | 先兆症状、头痛为主 |
| 周围性面瘫 | 额纹消失（中枢性面瘫额纹存在） |

## 7. 心力衰竭

### 7.1 诊断标准
- 心衰症状（呼吸困难、乏力、液体潴留）
- 客观证据：超声心动图异常（EF降低或舒张功能障碍）、BNP/NT-proBNP升高
- NYHA心功能分级

### 7.2 NYHA心功能分级

| 分级 | 表现 |
|------|------|
| I级 | 日常活动无症状 |
| II级 | 日常活动轻度受限，休息时无症状 |
| III级 | 日常活动明显受限，轻微活动即有症状 |
| IV级 | 休息时也有症状 |

### 7.3 治疗
- 利尿剂：减轻液体潴留
- ACEI/ARB/ARNI：改善预后
- β受体阻滞剂：改善预后
- 醛固酮受体拮抗剂：改善预后
- SGLT2抑制剂：改善预后（新四联）

### 7.4 BNP/NT-proBNP参考值

| 指标 | 排除心衰 | 诊断心衰 | 临界值 |
|------|---------|---------|--------|
| BNP | <100pg/mL | >400pg/mL | 100-400pg/mL |
| NT-proBNP（<50岁） | <125pg/mL | >450pg/mL | 125-450pg/mL |
| NT-proBNP（50-75岁） | <450pg/mL | >900pg/mL | 450-900pg/mL |
| NT-proBNP（>75岁） | <900pg/mL | >1800pg/mL | 900-1800pg/mL |

### 7.5 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| COPD | 长期吸烟史、桶状胸、FEV1/FVC<70% |
| 支气管哮喘 | 喘息、哮鸣音、过敏史 |
| 肺栓塞 | 突发呼吸困难、D-dimer升高 |
| 肝硬化腹水 | 腹水为主、肝功能异常 |
| 肾病综合征 | 大量蛋白尿、低蛋白血症 |

## 8. 慢性阻塞性肺疾病（COPD）

### 8.1 诊断标准
- 慢性咳嗽、咳痰、呼吸困难
- 吸烟史或危险因素暴露史
- 肺功能：吸入支气管舒张剂后FEV1/FVC<70%

### 8.2 GOLD分级

| 分级 | 肺功能（FEV1占预计值%） | 严重程度 |
|------|----------------------|---------|
| GOLD1 | ≥80% | 轻度 |
| GOLD2 | 50-79% | 中度 |
| GOLD3 | 30-49% | 重度 |
| GOLD4 | <30% | 极重度 |

### 8.3 ABE分组

| 组别 | 呼吸症状 | 急性加重史 | 推荐治疗 |
|------|---------|-----------|---------|
| A组 | 少 | 0-1次/年（无住院） | 支气管舒张剂 |
| B组 | 多 | 0-1次/年（无住院） | LABA+LAMA |
| E组 | 任何 | ≥2次/年或≥1次住院 | LABA+LAMA±ICS |

### 8.4 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 支气管哮喘 | 可逆性气流受限、过敏史 |
| 心力衰竭 | BNP升高、心超异常 |
| 支气管扩张 | 大量脓痰、CT支气管扩张 |
| 肺结核 | 低热盗汗、痰涂片阳性 |
| 间质性肺病 | 限制性通气障碍、HRCT异常 |

## 9. 支气管哮喘

### 9.1 诊断标准
1. 反复发作喘息、气急、胸闷或咳嗽
2. 发作时双肺可闻及散在或弥漫性哮鸣音，呼气相延长
3. 上述症状可经治疗缓解或自行缓解
4. 除外其他疾病引起的喘息、气急
5. 肺功能：支气管舒张试验FEV1改善≥12%且绝对值≥200mL

### 9.2 控制水平分级

| 分级 | 日间症状 | 夜间症状 | 活动受限 | FEV1 |
|------|---------|---------|---------|------|
| 控制 | ≤2次/周 | 无 | 无 | ≥80%预计值 |
| 部分控制 | >2次/周 | 有 | 有 | <80%预计值 |
| 未控制 | 每天都有 | 频繁 | 严重 | 严重下降 |

### 9.3 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| COPD | 吸烟史、不可逆气流受限 |
| 心源性哮喘 | BNP升高、心脏基础疾病 |
| 声带功能障碍 | 吸气性喘鸣、喉镜检查 |
| 气管异物 | 突发呛咳、局部呼吸音消失 |
| 过敏性肺炎 | 与环境暴露相关、CT磨玻璃影 |

## 10. 急性胰腺炎

### 10.1 诊断标准（满足以下3项中2项）
1. 腹痛（急性持续性上腹痛）
2. 血清淀粉酶或脂肪酶≥正常上限3倍
3. 影像学（CT/MRI/B超）符合胰腺炎改变

### 10.2 严重程度分级

| 分级 | 标准 | 死亡率 |
|------|------|--------|
| 轻症 | 无器官衰竭、无局部/全身并发症 | <1% |
| 中重症 | 一过性器官衰竭（<48h）或局部并发症 | <8% |
| 重症 | 持续器官衰竭（>48h） | 36-50% |

| LDH | >350IU/L | >400IU/L |
| AST | >250IU/L | >250IU/L |
| 入院48h内 | | |
| HCT下降 | >10% | >10% |
| BUN升高 | >1.8mmol/L | >1.8mmol/L |
| 钙离子 | <2mmol/L | <2mmol/L |
| 碱缺失 | >4mEq/L | >5mEq/L |
| 液体隔离 | >6000mL | >6000mL |
| **≥3分提示重症** | | |

### 10.4 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 消化性溃疡穿孔 | 溃疡病史、膈下游离气体 |
| 胆囊炎 | 右上腹痛、Murphy征、超声异常 |
| 肠梗阻 | 腹痛、呕吐、腹胀、停止排气排便 |
| 心肌梗死 | ECG异常、肌钙蛋白升高 |
| 异位妊娠 | 停经史、hCG阳性 |

## 11. 胆囊炎与胆石症

### 11.1 急性胆囊炎诊断
1. 右上腹痛（Murphy征阳性）
2. 发热
3. WBC升高
4. 超声：胆囊壁增厚>4mm、胆囊周围积液、结石

### 11.2 Tokyo分级

| 分级 | 严重程度 | 处理 |
|------|---------|------|
| I级（轻症） | 无器官功能障碍、无胆囊壁明显炎症 | 禁食+抗感染 |
| II级（中症） | WBC>18、可触及右上腹包块、症状>72h | 禁食+抗感染+早期手术 |
| III级（重症） | 器官功能障碍 | ICU+紧急手术 |

### 11.3 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 肝脓肿 | 高热、肝大压痛、CT液性暗区 |
| 胰腺炎 | 上腹痛放射至腰背、淀粉酶升高 |
| 阑尾炎 | 转移性右下腹痛 |
| 肾结石 | 绞痛放射至会阴、血尿 |
| 胃十二指肠溃疡 | 与进食相关、抗酸药缓解 |

## 13. 甲状腺功能亢进症

### 13.1 诊断标准
1. 甲亢临床表现（高代谢症状、甲状腺肿大、眼征）
2. TSH降低（<0.1mU/L），FT4升高
3. TRAb阳性（Graves病）

### 13.2 实验室检查

| 指标 | 正常参考值 | 甲亢 |
|------|-----------|------|
| TSH | 0.35-4.94mU/L | <0.1mU/L |
| FT3 | 2.63-5.70pmol/L | 升高 |
| FT4 | 9.01-19.05pmol/L | 升高 |
| TRAb | <1.75IU/L | 升高（Graves病） |
| TPOAb | <34IU/mL | 可升高 |

### 13.3 Graves病治疗

| 治疗方法 | 适应证 | 优点 | 缺点 |
|---------|--------|------|------|
| 抗甲状腺药物 | 轻中度甲亢、年轻 | 无创 | 疗程长、复发率高 |
| 放射性碘-131 | 药物过敏、复发 | 根治 | 可能甲减 |
| 手术 | 甲状腺肿大、压迫 | 快速 | 手术风险 |

### 13.4 甲亢危象诊断

| 指标 | 标准 |
|------|------|
| 体温 | >39°C |
| 心率 | >160次/分 |
| 消化道 | 恶心呕吐腹泻 |
| 神经系统 | 谵妄、昏迷 |
| 循环 | 心衰、休克 |

### 13.5 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 亚急性甲状腺炎 | 甲状腺疼痛、ESR升高、摄碘率降低 |
| 桥本甲状腺炎 | 慢性病程、TPOAb高滴度 |
| 嗜铬细胞瘤 | 阵发性高血压、儿茶酚胺升高 |

## 15. 痛风

### 15.1 诊断标准（2015 ACR/EULAR）

**准入标准（必须）：** 至少1次外周关节或滑囊肿胀、疼痛或压痛

**充分标准（不需要评分即确诊）：** 受累关节或滑囊的滑液或痛风石中检测到尿酸单钠（MSU）晶体

### 15.2 临床分期

| 分期 | 特点 | 处理 |
|------|------|------|
| 无症状高尿酸血症 | 血尿酸>420μmol/L、无关节症状 | 生活方式干预 |
| 急性痛风性关节炎 | 关节红肿热痛、急性发作 | NSAIDs/秋水仙碱 |
| 痛风间歇期 | 发作间期、血尿酸高 | 开始降尿酸治疗 |
| 慢性痛风石性痛风 | 痛风石形成、关节破坏 | 强化降尿酸+手术 |

### 15.3 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 假性痛风 | 老年人、膝关节为主、焦磷酸钙沉积 |
| 化脓性关节炎 | 发热、关节液培养阳性 |
| 类风湿关节炎 | 对称性小关节、晨僵>1h |
| 反应性关节炎 | 肠道/泌尿系感染后 |
| 蜂窝织炎 | 皮肤红肿热痛、无滑膜炎 |

## 16. 急性心肌梗死

### 16.1 诊断标准（第四版心肌梗死定义）

### 16.3 鉴别诊断

## 18. 深静脉血栓形成（DVT）

### 18.1 诊断要点
- 单侧下肢肿胀、疼痛

- Homans征阳性（背屈足踝时小腿后方疼痛）

### 18.2 Wells评分

| 全腿肿胀 | +1 |
| 小腿肿胀>3cm | +1 |
| 凹陷性水肿（患侧重） | +1 |

### 18.4 鉴别诊断

## 19. 肺栓塞

### 19.1 诊断流程
1. 临床怀疑（突发呼吸困难、胸痛、咯血）

3. D-dimer检测（低/中概率）
4. CTPA确诊

### 19.2 Wells评分

| 指标 | 分值 |
|------|------|

| PE为第一诊断/不能排除 | +3 |
| 心率>100次/分 | +1.5 |
| 制动>3天或4周内手术 | +1.5 |
| 既往DVT/PE | +1.5 |
| 咯血 | +1 |
| 恶性肿瘤 | +1 |
| **≥4分：高概率** | |

### 19.3 治疗

| 情况 | 处理 |
|------|------|
| 低危（PESI I-II） | 可门诊抗凝 |
| 中危（PESI III-IV） | 住院抗凝 |
| 高危（休克/低血压） | 溶栓/取栓 |
| 大面积PE | 系统溶栓/外科取栓 |

### 19.4 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 急性冠脉综合征 | ECG变化、肌钙蛋白升高 |
| 气胸 | 患侧呼吸音减弱 |
| 肺炎 | 发热、咳痰、白细胞升高 |
| 心包炎 | 心包摩擦音、ECG弥漫ST抬高 |

---

## 20. 慢性肾脏病（CKD）

### 20.1 诊断标准
- 肾脏损伤标志物异常（蛋白尿、血尿、影像学异常）持续>3个月
- 和/或GFR<60mL/(min·1.73m²)持续>3个月

### 20.2 CKD分期

| 分期 | GFR [mL/(min·1.73m²)] | 描述 |
|------|----------------------|------|
| G1 | ≥90 | 肾功能正常 |
| G2 | 60-89 | 轻度下降 |
| G3a | 45-59 | 轻中度下降 |
| G3b | 30-44 | 中重度下降 |
| G4 | 15-29 | 重度下降 |
| G5 | <15 | 肾衰竭 |

### 20.3 蛋白尿分级

| 分级 | 尿蛋白排泄率 | 临床意义 |
|------|------------|---------|
| 正常 | <30mg/24h | 无蛋白尿 |
| 微量白蛋白尿 | 30-300mg/24h | 早期肾损害 |
| 临床蛋白尿 | >300mg/24h | 明显肾损害 |
| 肾病范围蛋白尿 | >3500mg/24h | 肾病综合征 |

### 20.4 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 急性肾损伤 | 短期内GFR下降、可逆 |
| 糖尿病肾病 | 糖尿病病史、视网膜病变 |
| 高血压肾损害 | 长期高血压、左室肥厚 |
| 慢性肾炎 | 血尿、蛋白尿、免疫异常 |
| 梗阻性肾病 | 肾积水、结石、前列腺增生 |

---

## 21. 缺铁性贫血

### 21.1 诊断标准
- 血红蛋白降低：男性<130g/L，女性<120g/L
- 血清铁蛋白<12μg/L（金标准）
- MCV<80fL、MCH<27pg、MCHC<320g/L
- 铁剂治疗有效

### 21.2 实验室检查

| 指标 | 正常参考值 | 缺铁性贫血 |
|------|-----------|-----------|
| Hb | M130/F120g/L | 降低 |
| MCV | 80-100fL | <80fL |
| 血清铁 | 11-30μmol/L | 降低 |
| 血清铁蛋白 | 12-300μg/L | <12μg/L |
| TIBC | 45-72μmol/L | 升高 |
| 转铁蛋白饱和度 | 20-50% | <20% |

### 21.3 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 地中海贫血 | 靶形红细胞、血红蛋白电泳异常 |
| 慢性病性贫血 | 铁蛋白正常或升高、TIBC降低 |
| 巨幼细胞性贫血 | MCV>100fL、叶酸/B12降低 |
| 再生障碍性贫血 | 全血细胞减少、骨髓增生低下 |

## 22. 急性胃肠炎

### 22.1 诊断要点
- 腹泻（≥3次/日稀便或水样便）
- 和/或呕吐
- 常伴腹痛、发热

### 22.2 脱水程度评估

| 程度 | 体重下降 | 临床表现 | 处理 |
|------|---------|---------|------|
| 轻度 | <5% | 口渴、尿量略减 | 口服补液 |
| 中度 | 5-10% | 皮肤弹性差、眼窝凹陷 | 口服/静脉补液 |
| 重度 | >10% | 休克征象 | 紧急静脉补液 |

### 22.3 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 食物中毒 | 群体发病、共同进食史 |
| 阑尾炎早期 | 转移性右下腹痛 |
| 炎症性肠病 | 慢性病程、血便 |
| 肠梗阻 | 停止排气排便、腹胀 |

---

## 23. 带状疱疹

### 23.1 诊断要点
- 沿单侧皮节分布的簇集性水疱
- 前驱症状（疼痛、瘙痒）
- 神经痛
- 免疫力低下者可播散

### 23.2 治疗

| 药物 | 剂量 | 疗程 |
|------|------|------|
| 阿昔洛韦 | 800mg q4h×5天 | 5-7天 |
| 泛昔洛韦 | 500mg tid×7天 | 7天 |
| 加巴喷丁 | 300-3000mg/d | 神经痛 |

### 23.3 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 单纯疱疹 | 复发性、簇集小水疱、非皮节分布 |
| 接触性皮炎 | 接触史、瘙痒为主 |
| 虫咬皮炎 | 散在丘疹、瘙痒 |

## 24. 颈椎病

### 24.1 分型与诊断

| 类型 | 主要表现 | 关键检查 |
|------|---------|---------|
| 神经根型 | 上肢放射痛、麻木 | MRI神经根受压 |
| 脊髓型 | 四肢无力、踩棉花感 | MRI脊髓受压 |
| 椎动脉型 | 头晕、猝倒 | MRA椎动脉受压 |
| 交感型 | 头晕、心悸、视物模糊 | 排除其他诊断 |

### 24.2 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 肩周炎 | 肩关节活动受限为主 |
| 腕管综合征 | 手指麻木、Tinel征 |
| 胸廓出口综合征 | 上肢缺血表现 |
| 脊髓肿瘤 | 进行性加重、MRI占位 |
