# 常见药物相互作用临床指南（完整版）

## 1. 概述

药物相互作用是指两种或两种以上药物同时或先后使用时，由于药效学或药动学的改变，导致药物效应增强或减弱，甚至产生不良反应的现象。药物相互作用是临床用药安全的重要隐患，据WHO统计，药物不良反应中约有6%-30%与药物相互作用有关。在住院患者中，接受5种以上药物的患者发生药物相互作用的风险高达50%，而接受10种以上药物的患者风险高达80%。

### 1.1 药物相互作用分类

#### 药动学相互作用
- **吸收阶段**：改变胃肠道pH值、形成络合物、影响胃肠运动、改变肠道菌群
- **分布阶段**：竞争蛋白结合位点、改变组织分布、影响血脑屏障通透性
- **代谢阶段**：酶诱导（加速代谢）、酶抑制（减慢代谢）、首过效应改变
- **排泄阶段**：改变肾小管分泌、影响尿液pH、改变胆汁排泄

#### 药效学相互作用
- **协同作用**：相加作用、增强作用、增敏作用
- **拮抗作用**：竞争性拮抗、非竞争性拮抗、生理性拮抗、化学性拮抗

### 1.2 相互作用严重程度分级

| 分级 | 定义 | 临床处理 |
|------|------|---------|
| 5级（禁忌联用） | 可能致死或严重损害 | 绝对禁止联用 |
| 4级（避免联用） | 需要调整剂量或更换药物 | 避免联用，必须联用需严密监测 |
| 3级（谨慎联用） | 可能增加不良反应风险 | 谨慎联用，加强监测 |
| 2级（注意联用） | 轻微相互作用 | 一般无需调整，注意观察 |
| 1级（安全联用） | 无明显相互作用 | 可安全联用 |

### 1.3 药物代谢酶系统概述

| CYP酶亚型 | 底物药物（示例） | 抑制剂 | 诱导剂 |
|-----------|-----------------|--------|--------|
| CYP3A4 | 环孢素、他克莫司、辛伐他汀、阿托伐他汀、硝苯地平、咪达唑仑 | 酮康唑、伊曲康唑、利托那韦、葡萄柚汁 | 利福平、苯妥英钠、卡马西平、圣约翰草 |
| CYP2D6 | 美托洛尔、可待因、曲马多、氟西汀、帕罗西汀 | 氟西汀、帕罗西汀、奎尼丁 | 利福平（弱诱导） |
| CYP2C9 | 华法林（S-）、苯妥英钠、洛沙坦、布洛芬 | 氟康唑、胺碘酮、甲硝唑 | 利福平、苯巴比妥 |
| CYP2C19 | 奥美拉唑、氯吡格雷、地西泮 | 奥美拉唑、埃索美拉唑、氟伏沙明 | 利福平 |
| CYP1A2 | 茶碱、咖啡因、氯氮平 | 环丙沙星、氟伏沙明、氟喹诺酮类 | 吸烟、奥美拉唑 |

### 1.4 转运体介导的相互作用

| 转运体 | 位置 | 底物药物 | 抑制剂 | 临床意义 |
|--------|------|---------|--------|---------|
| P-糖蛋白（P-gp） | 小肠、肾、肝、血脑屏障 | 地高辛、环孢素、利伐沙班、达比加群、维拉帕米 | 维拉帕米、胺碘酮、决奈达隆、利托那韦 | 升高底物药物血药浓度 |
| OATP1B1 | 肝脏 | 他汀类、甲氨蝶呤、沙坦类 | 环孢素、吉非贝齐 | 升高药物肝内蓄积风险 |
| OCT2 | 肾脏 | 二甲双胍、西咪替丁、顺铂 | 西咪替丁 | 减少肾排泄，升高血药浓度 |

## 2. 抗凝血药物相互作用

### 2.1 华法林（Warfarin）相互作用

华法林是维生素K拮抗剂，治疗窗窄，是药物相互作用最复杂的药物之一。其代谢主要依赖CYP2C9（S-华法林）和CYP3A4（R-华法林）。

| 氟伐他汀 | 抑制CYP2C9 | 中等 | 升高20-40% | 监测INR |
| 奥美拉唑 | 抑制CYP2C19 | 轻微 | 升高10-20% | 一般无需调整 |

| 拉莫三嗪 | 抑制CYP2C9 | 轻微-中等 | 升高10-20% | 监测INR |
| 氟伐他汀 | 抑制CYP2C9 | 中等 | 升高20-40% | 监测INR |

### 2.2 新型口服抗凝药（NOACs）相互作用

#### 利伐沙班（Rivaroxaban）
- CYP3A4和P-gp双重底物
- 避免联用：酮康唑、伊曲康唑、利托那韦（强效CYP3A4/P-gp抑制剂）
- 谨慎联用：克拉霉素、红霉素、环孢素
- 联用抗血小板药物：出血风险增加
- 与丙戊酸联用：丙戊酸可能降低利伐沙班暴露量
- 与苯妥英钠/卡马西平联用：可能降低利伐沙班效果

#### 达比加群（Dabigatran）
- P-gp底物，不经CYP代谢
- 避免联用：维拉帕米（需减量）、奎尼丁、决奈达隆
- 联用胺碘酮、氯吡格雷：出血风险增加
- 胺碘酮使达比加群浓度升高约50%（口服给药），需减少达比加群剂量至150mg bid
- 维拉帕米使达比加群浓度升高约120%（口服给药），需减量
- 与质子泵抑制剂联用：可能降低达比加群吸收（但临床意义有限）

#### 阿哌沙班（Apixaban）
- CYP3A4和P-gp双重底物
- 强效抑制剂（酮康唑、伊曲康唑）：禁用或减量
- 弱效抑制剂（氟康唑、红霉素）：谨慎联用
- 联用NSAIDs：出血风险增加

#### 艾多沙班（Edoxaban）
- P-gp底物，CYP3A4影响小
- 与P-gp抑制剂联用需减量
- 与胺碘酮、维拉帕米、决奈达隆联用：减量至30mg qd
- 避免与环孢素联用

### 2.3 抗血小板药物相互作用

#### 阿司匹林
- 与布洛芬联用：可能降低心血管保护作用（布洛芬竞争COX-1结合位点）
- 与华法林联用：出血风险显著增加
- 与SSRIs联用：出血风险增加2-4倍
- 与甲氨蝶呤联用：增加甲氨蝶呤毒性（减少肾排泄）
- 与ACEI联用：可能减弱降压效果
- 与噻氯匹定联用：协同抗血小板，出血风险增加

#### 氯吡格雷
- CYP2C19底物，前体药物
- 奥美拉唑/埃索美拉唑：抑制CYP2C19，降低氯吡格雷效果（避免联用）
- 兰索拉唑、泮托拉唑：影响较小，可选用
- 与华法林联用：出血风险显著增加
- 与萘普生联用：出血风险增加
- 与苯妥英钠联用：可能降低苯妥英钠效果
- 阿瑞匹坦：降低氯吡格雷效果约40%

#### 替格瑞洛（Ticagrelor）
- CYP3A4底物，P-gp底物
- 避免联用强效CYP3A4抑制剂（酮康唑、伊曲康唑、利托那韦）
- 避免联用强效CYP3A4诱导剂（利福平、苯妥英钠、卡马西平）
- 与辛伐他汀联用：辛伐他汀剂量不超过40mg
- 与洛伐他汀联用：避免联用
- 与地高辛联用：地高辛浓度升高约10%
- 与环孢素联用：需避免联用
- 与他克莫司联用：需监测他克莫司浓度

## 3. 心血管药物相互作用

### 3.1 β受体阻滞剂

| 联用药物 | 相互作用 | 临床表现 | 处理建议 |
|----------|---------|---------|---------|
| 维拉帕米/地尔硫卓 | 协同抑制心肌传导 | 严重心动过缓、传导阻滞 | 禁忌联用或极度谨慎 |
| 地高辛 | 协同减慢心率 | 心动过缓 | 监测心率和地高辛浓度 |
| 胰岛素/磺脲类 | 掩盖低血糖症状 | 低血糖不易察觉 | 加强血糖监测 |
| NSAIDs | 减弱降压效果 | 血压控制不佳 | 避免长期联用 |
| 可乐定 | 协同降压 | 低血压、心动过缓 | 停可乐定时需逐渐减量 |
| 肾上腺素 | α受体阻断 | 高血压危象 | 避免联用 |
| 氟烷/七氟烷 | 增强麻醉药心脏抑制 | 低血压、心动过缓 | 麻醉前评估 |
| 钙剂 | 拮抗β阻滞剂的降压作用 | 降压效果减弱 | 监测血压 |
| 丙咪嗪 | 协同降压 | 体位性低血压 | 监测血压 |
| 胰高血糖素 | 协同降压 | 低血压 | 谨慎联用 |
| 胍乙啶 | 协同降压 | 严重低血压 | 避免联用 |

### 3.2 ACEI/ARB类

| 联用药物 | 相互作用机制 | 风险 | 监测指标 |
|----------|-------------|------|---------|
| 保钾利尿剂（螺内酯） | 协同升高血钾 | 高钾血症 | 血钾、肾功能 |
| NSAIDs | 抑制前列腺素合成 | 降压效果减弱+肾损伤 | 血压、肌酐、血钾 |
| 补钾制剂 | 增加钾摄入 | 高钾血症 | 血钾 |
| 锂盐 | 减少锂排泄 | 锂中毒 | 锂浓度 |
| 阿替洛尔 | 协同降压 | 低血压 | 血压 |
| 甲氨蝶呤 | 减少肾排泄 | 甲氨蝶呤毒性 | 甲氨蝶呤浓度 |
| 别嘌醇 | 增加超敏反应风险 | 皮肤过敏反应 | 观察皮疹 |
| 糖皮质激素 | 对抗降压效果 | 血压升高 | 血压 |

#### ACEI特有相互作用
- 与阿司匹林联用：可能减弱ACEI的心血管保护作用（阿司匹林抑制前列腺素合成）
- 与西地那非联用：严重低血压（禁忌联用）
- 与血管紧张素疫苗联用：避免联用

#### ARB特有相互作用
- 与沙库巴曲缬沙坦联用：避免与ACEI联用（需洗脱36小时）
- 与阿利吉仑联用（双肾素系统阻断）：增加高钾血症和肾损伤风险

### 3.3 钙通道阻滞剂（CCBs）

#### 二氢吡啶类（氨氯地平、硝苯地平）
- CYP3A4底物，葡萄柚汁可显著升高血药浓度
- 与β受体阻滞剂联用：降压效果增强，注意低血压
- 与地高辛联用：地高辛浓度升高30-50%
- 与辛伐他汀联用：辛伐他汀浓度显著升高，需限制辛伐他汀剂量≤20mg
- 与西咪替丁联用：CCB浓度升高（氨氯地平影响小）
- 与苯妥英钠联用：双向相互作用
- 与环孢素联用：环孢素浓度升高
- 与西地那非联用：严重低血压风险
- 与利福平联用：CCB效果减弱
- 与芬太尼联用：严重低血压

#### 非二氢吡啶类（维拉帕米、地尔硫卓）
- CYP3A4底物
- 与β受体阻滞剂联用：禁忌（严重心动过缓）
- 与地高辛联用：地高辛浓度升高50-100%
- 与他汀类联用：增加横纹肌溶解风险
- 与卡马西平联用：卡马西平浓度升高
- 与茶碱联用：茶碱浓度升高
- 与丙戊酸联用：丙戊酸浓度升高
- 与地高辛联用：需减少地高辛剂量50%
- 与苯巴比妥联用：维拉帕米效果减弱
- 与奎尼丁联用：奎尼丁浓度升高，需避免联用

| 华法林 | 竞争蛋白结合 | INR可能升高 | 监测INR |
| 氟康唑 | 抑制CYP2C9 | 横纹肌溶解 | 监测CK |

## 11. 特殊人群用药注意事项

### 11.1 老年患者（65岁以上）
- 多重用药风险高，平均用药5-9种
- 肝肾功能减退，药物清除率下降30-50%
- 对中枢神经系统药物敏感性增加
- Beers标准：列出老年人应避免使用的药物

#### 老年人应避免使用的药物（Beers标准精选）

| 药物 | 风险 | 替代方案 |
|------|------|---------|
| 长效苯二氮卓类（地西泮、氟西泮） | 谵妄、跌倒、骨折 | 短效苯二氮卓或非药物治疗 |
| 抗胆碱药（苯海索、氯苯那敏） | 谵妄、便秘、尿潴留 | 选择抗胆碱作用小的药物 |
| 第一代抗组胺药（苯海拉明） | 镇静、谵妄 | 第二代抗组胺药 |
| 巴比妥类 | 过度镇静、跌倒 | 苯二氮卓类或非药物治疗 |
| 甲基多巴 | 肝毒性、抑郁 | ACEI/ARB |
| 哌替啶 | 神经毒性、代谢物蓄积 | 吗啡（注意减量） |
| 长效NSAIDs | 胃肠道出血、肾损伤 | 选择性COX-2抑制剂或对乙酰氨基酚 |
| 胺碘酮 | 甲状腺毒性、肺毒性 | 其他抗心律失常药 |
| 强心苷（地高辛） | 中毒风险增加 | 低剂量维持 |
| 螺内酯（>25mg） | 高钾血症 | 其他利尿剂 |

#### 老年患者用药原则
1. 从最小有效剂量开始
2. 一次不超过5种新药
3. 定期审查用药清单
4. 避免使用抗胆碱能药物
5. 监测肾功能并据此调整剂量
6. 注意药物-药物和药物-疾病相互作用

### 11.2 肝功能不全
- Child-Pugh分级指导用药
- A级（5-6分）：多数药物可正常使用
- B级（7-9分）：需调整剂量
- C级（10-15分）：避免使用肝毒性药物

| 阿普唑仑 | 减量25% | 减量50% | 禁用 | 避免使用 |
| 咪达唑仑 | 减量50% | 减量75% | 禁用 | 避免使用 |
| 吗啡 | 减量25% | 减量50% | 避免使用 | 换用其他镇痛药 |
| 普萘洛尔 | 减量 | 禁用 | 禁用 | 换用其他β阻滞剂 |
| 茶碱 | 正常剂量 | 减量25% | 减量50% | 监测浓度 |
| 维拉帕米 | 减量 | 禁用 | 禁用 | 换用其他降压药 |

### 11.3 肾功能不全

- CrCl 50-80：部分药物需减量

#### 妊娠期药物安全性分级（FDA旧分类参考）

| 分级 | 定义 | 代表药物 |

### 11.6 儿童用药
- 儿童肝药酶系统发育不完善（CYP3A4在2-4岁达到成人水平）
- 肾功能在2岁左右达到成人水平
- 药物分布容积与成人不同
- 禁忌使用：氟喹诺酮类（影响软骨发育）、四环素类（牙齿变色）、阿司匹林（Reye综合征）
- 对镇静催眠药和阿片类药物更敏感
- 需按体重或体表面积精确计算剂量

## 14. 药物相互作用管理流程

### 14.1 处方审核流程

```
患者用药列表
    ↓
药物相互作用筛查（DDI数据库）
    ↓
发现相互作用？
    ├─ 否 → 继续处方
    └─ 是 → 评估严重程度
              ├─ 1-2级 → 记录，继续用药
              ├─ 3级 → 加强监测
              ├─ 4级 → 考虑替代方案或调整剂量
              └─ 5级 → 绝对避免联用，更换药物
```
