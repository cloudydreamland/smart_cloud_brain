## RabbitMQ / KingbaseES 搜索验收

| 场景 | 操作 | 期望 |
|---|---|---|
| Outbox 可靠投递 | RabbitMQ 暂停时审核处方 | 主业务成功，`outbox_event` 保持待投递或失败状态 |
| 事件恢复 | RabbitMQ 恢复后等待定时任务 | 事件投递成功并标记 `SENT` |
| 金仓搜索 | 管理端新增或修改知识库、药品、Prompt 后搜索 | 搜索接口直接从 KingbaseES 返回最新数据 |
| 风险通知 | AI 处方审核返回 `HIGH` 或 `MEDIUM` | `notification-service` 消费事件，写通知并推送 WebSocket |
